# Task 3: Cat vs Dog Image Classification using SVM

## Objective

Implement a Support Vector Machine (SVM) classifier to classify images of cats and dogs.

## Dataset

Microsoft Cats and Dogs Dataset

## Technologies Used

* Python
* OpenCV
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* HOG (Histogram of Oriented Gradients)

## Project Workflow

1. Download and extract dataset
2. Image preprocessing
3. Grayscale conversion
4. HOG feature extraction
5. Train-test split
6. Feature scaling
7. SVM model training
8. Performance evaluation

## Results

* Accuracy: 73.5%
* Classification Report
* Confusion Matrix
* Sample Predictions

## Conclusion

The model successfully classified cat and dog images using HOG features and an SVM classifier. The project demonstrates how traditional machine learning techniques can be applied to image classification tasks.
